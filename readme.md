## Linux

### Dependencies

*   CMake
*   Internet connectie (Voor alle libraries welke cmake voor je download)

```bash
cd magic_tool
mkdir build
cd build

cmake ..
make
```

# Om het programma te gebruiker

```bash
./magic_tool
```
